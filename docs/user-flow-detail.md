# Once Upon Us - 상세 사용자 플로우 (v2: 모듈형 스토리 블록)

> 업데이트: 2026-04-03
> 고정 9단계 위저드 → 모듈형 스토리 블록 시스템으로 전환 완료

## 전체 흐름 요약

```
랜딩페이지 → Step 0~2 (준비/사진/스토리 선택) → 오프닝 → 선택 블록 5개 → 아웃트로 → 최종 검토 → 영상 완성
```

---

## Step 0: 준비물 확인 — "이것만 준비하면 돼요"

> 입력 없음 (안내 페이지)

- 체크리스트 3개:
  - 신랑 얼굴 사진 (정면, 밝은 조명 1장)
  - 신부 얼굴 사진 (정면, 밝은 조명 1장)
  - 커플 추억 사진 (섹션별 최소 1장씩)
- 버튼: **"준비됐어요, 시작하기"** / "전문가에게 생성 문의하기"

---

## Step 1: 캐릭터 생성 — "두 분의 사진을 올려주세요"

| 입력 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 신랑 이름 | 텍스트 | **필수** | placeholder: "신랑 이름" |
| 신부 이름 | 텍스트 | **필수** | placeholder: "신부 이름" |
| 신랑 사진 | 사진 업로드 | 선택 | 탭하면 샘플 사용 가능 |
| 신부 사진 | 사진 업로드 | 선택 | 탭하면 샘플 사용 가능 |

> 완료 후 **로그인 게이트 모달** → Google / Kakao 소셜 로그인 또는 이메일 로그인

---

## Step 2: 스토리 선택 — "어떤 스토리를 만들어볼까요?"

### 옵션 A: 템플릿 선택 (4종)

| 템플릿 | 자동 구성 블록 5개 | 톤 |
|--------|-------------------|-----|
| 💎 클래식 러브스토리 | 첫 만남 → 사랑에 빠진 순간 → 프로포즈 → 서로에게 하는 말 → 부모님 감사 | 정석, 감동 |
| 🏡 우리만의 일상 | 우리는 이런 사람 → 일상의 행복 → 함께한 여행 → 반려동물/취미 → 미래의 약속 | 따뜻, 소소 |
| 🎬 드라마틱 스토리 | 첫인상 → 첫 데이트 → 위기와 극복 → 프로포즈 → 서로에게 하는 말 | 극적, 영화같은 |
| 👨‍👩‍👧‍👦 가족 중심 | 어린 시절 → 첫 만남 → 부부가 되기로 → 부모님 감사 → 미래의 약속 | 한국식, 전통 |

### 옵션 B: "직접 고를래요"

15개 블록 풀에서 5개를 선택하고 순서를 조정:

| 카테고리 | 블록 | 사진 필수 |
|----------|------|-----------|
| 소개 | 👫 우리는 이런 사람 | - |
| 소개 | 💒 어린 시절 | 1~2장 |
| 만남 | 🤝 첫 만남 | 1~3장 |
| 만남 | ✨ 첫인상 | - |
| 연애 | 💕 첫 데이트 | 1~3장 |
| 연애 | 💘 사랑에 빠진 순간 | 1~3장 |
| 연애 | ✈️ 함께한 여행 | 1~3장 |
| 연애 | ☕ 일상의 행복 | 1~3장 |
| 연애 | 🌧️ 위기와 극복 | 1~2장 |
| 특별 | 🐾 반려동물/취미 | 1~3장 |
| 전환 | 💍 프로포즈 | 1~2장 |
| 전환 | 🎊 부부가 되기로 | 1~2장 |
| 가족 | 🙏 부모님 감사 | 0~2장 |
| 약속 | 💌 서로에게 하는 말 | - |
| 약속 | 🌅 미래의 약속 | 0~1장 |

> 5개 선택 후 순서 조정 → "이 구성으로 시작하기"

---

## Step 3: 오프닝 인사 (고정)

| 입력 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 인사 문장 | textarea | 선택 | placeholder: "안녕하세요, {신랑}과 {신부}입니다." / 최대 120자 |

---

## Step 4~8: 선택한 블록 5개 (동적)

각 블록은 **BlockEditor**가 정의에 따라 자동 렌더링:
- 블록별 입력 필드 (텍스트, textarea, 칩 선택)
- 사진 업로드 (블록별 최소/최대 장수)
- 나레이션 자동 생성 + 수정 가능 (최대 150자)

---

## Step 9: 아웃트로 - 감사합니다 (고정)

| 입력 | 타입 | 필수 | 설명 |
|------|------|------|------|
| 감사 메시지 | textarea | 선택 | 기본값: "와주셔서 감사합니다." / 최대 120자 |
| 엔딩 사진 | 파일 업로드 | 선택 | 최대 1장 |

---

## Step 10: 최종 검토

동적 체크리스트:
- 오프닝 (선택)
- 선택한 블록 5개 (사진 필수 블록은 검증)
- 아웃트로 (선택)

> 모든 필수 사진 통과 시 **"영상 완성하기"** 버튼 활성화

---

## 영상 생성 이후 흐름

```
"영상 완성하기" 클릭
  → AI 스토리 생성 (Gemini) - 나레이션 기반
  → AI 이미지 생성 (FLUX.1) - 디즈니/지브리 스타일
  → AI 나레이션 생성 (ElevenLabs TTS)
  → 영상 렌더링 (Remotion) - HD/FHD/4K MP4
  → 미리보기 → 결제 (토스페이먼츠) → 다운로드
```
