## Phase 1: MVP 완성 (필수)

> 목표: 첫 고객이 결제하고 영상을 받을 수 있는 상태

### 1.1 인증 시스템 완성
- [ ] Kakao OAuth 프론트엔드 연동
- [ ] Google OAuth 프론트엔드 연동
- [ ] 로그인 상태 관리 (세션/JWT)
- [ ] 로그인 게이트 모달에서 실제 인증 처리
- [ ] 회원가입 페이지 API 연동

### 1.2 위저드 -> API 연동
- [ ] Step1 사진 업로드: S3 presigned URL 사용 (현재 demo mode)
- [ ] Step9 최종 제출: 프로젝트 생성 API에 전체 데이터 전송
- [ ] 생성 진행 페이지(`/projects/[id]/generating`)로 리다이렉트 연결
- [ ] 생성 완료 페이지(`/projects/[id]/complete`)에서 영상 확인

### 1.3 결제 플로우 연결
- [ ] 생성 완료 후 체크아웃 페이지로 이동
- [ ] 토스페이먼츠 결제창 호출
- [ ] 결제 성공 후 영상 다운로드 링크 제공
- [ ] 결제 실패/취소 처리

### 1.4 대시보드 API 연동
- [ ] 내 프로젝트 목록 조회 API 연결
- [ ] 프로젝트 상태별 표시 (생성중/완료/결제완료)
- [ ] 프로젝트 상세 보기 -> 편집/결제/다운로드

### 1.5 배포 & 인프라
- [ ] PostgreSQL 프로덕션 DB 설정 (Supabase 또는 AWS RDS)
- [ ] Redis 프로덕션 설정 (Upstash 또는 AWS ElastiCache)
- [ ] AWS S3 버킷 설정 + CORS
- [ ] API 서버 배포 (Railway / Fly.io / AWS ECS)
- [ ] 환경 변수 프로덕션 설정
- [ ] 도메인 연결 (onceuponus.com)
