# Once Upon Us - 서비스 기획서

## 1. 서비스 개요

**서비스명:** Once Upon Us (원스 어폰 어스)

**한 줄 소개:** AI가 커플의 사진과 이야기로 만드는 세상에 하나뿐인 식전 애니메이션 영상

**핵심 가치:** 모든 커플이 전문가 없이도 감동적인 웨딩 애니메이션을 만들 수 있도록

---

## 2. 타겟 사용자

### 주요 타겟
- **예비 신혼부부** (25-35세)
- 식전 영상을 원하지만 전문 업체 비용(50-200만원)이 부담스러운 커플
- 직접 만들고 싶지만 영상 편집 기술이 없는 커플

### 보조 타겟
- 웨딩 플래너 / 스튜디오 (B2B)
- 프로포즈 / 기념일 영상 제작 수요

---

## 3. 핵심 기능

### 3.1 AI 스토리 생성
- Google Gemini 기반 4막 구조 자동 스토리 생성
- 커플의 입력(만남, 연애, 결심, 감사)을 기반으로 개인화된 나레이션 작성
- 한국어/영어/일본어 지원

### 3.2 AI 캐릭터 & 장면 생성
- HuggingFace FLUX.1을 활용한 디즈니/지브리 스타일 일러스트 생성
- 커플 얼굴 사진 기반 캐릭터 생성
- 8가지 애니메이션 스타일 지원 (동화, 디즈니, 지브리, 수채화, 유화, 일러스트, 팝아트, 코믹)

### 3.3 AI 나레이션
- ElevenLabs TTS로 감정이 담긴 나레이션 자동 생성
- 남성/여성 내레이터 선택 가능

### 3.4 영상 렌더링
- Remotion 기반 고품질 MP4 출력
- HD(720p) / FHD(1080p) / 4K 해상도 지원
- 16:9 비율, 식전 상영 최적화

### 3.5 실시간 미리보기
- 위저드 과정에서 실시간 목업 플레이어 제공
- 예시 데이터 / 내 입력 데이터 토글

## 5. 비즈니스 모델

### 요금제

| 플랜 | 가격 | 포함 사항 |
|------|------|----------|
| **Basic** | ~~99,000원~~ 49,000원 | HD 영상 1편, 기본 스타일 1종, 수정 1회 |
| **Standard** | 199,000원 | FHD 영상 1편, 스타일 3종 선택, 수정 3회, BGM 선택 |
| **Premium** | 299,000원 | 4K 영상 1편, 전체 스타일, 무제한 수정, 프리미엄 BGM, 초대장 링크 |

### B2B
- 웨딩 플래너/스튜디오 대량 할인
- 맞춤 브랜딩 옵션
- API 연동 가능

### 수익 구조
- 영상 1건당 마진: 원가(AI API 비용) 약 3,000-5,000원 vs 판매가 49,000-299,000원
- 목표 마진율: 85-95%

## 6. 결제 시스템

- **PG사:** 토스페이먼츠 (Toss Payments)
- **지원 수단:** 카드, 간편결제(토스페이, 카카오페이 등)
- **플로우:** 주문 생성 -> 토스 결제창 -> 결제 확인 -> 영상 최종 완성

## 8. 경쟁 우위

1. **가격 경쟁력** - 전문 업체 대비 1/10~1/40 가격
2. **속도** - 주문 후 수분 내 영상 완성 (전문 업체: 1-4주)
3. **개인화** - 커플의 실제 이야기 기반, 기성품 아님
4. **접근성** - 기술 지식 없이도 위저드만 따라가면 완성
5. **다양한 스타일** - 8종 애니메이션 스타일 선택 가능

## 9. 시장 규모

- 한국 연간 결혼 건수: 약 19만 건 (2024 기준)
- 식전 영상 제작 수요: 약 60% (11.4만 건)
- 목표 시장 점유율 (1년차): 1% = 1,140건
- 예상 연매출 (1년차): 1,140건 x 평균 15만원 = 약 1.7억원
